eu.femboy

This page resolves through a delegated TLD registry, not through the Federate root zone.

The chain that verified this page:

  1. Federate Root Key signed the .femboy TLD record (the delegation).
  2. The .femboy operator key signed the TLD registry and this domain record.
  3. The domain owner key signed the site manifest.
  4. Every content block matched its BLAKE3 hash.

The root decides which TLDs exist. The operator decides which domains exist under its TLD. The owner decides the content. No layer can forge the one below it.

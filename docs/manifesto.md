# The Federate Manifesto

**Federate Network is a human web protocol.**

A human web, built by people. No feeds. No scraping. No AI training by default.

## What Federate is not

- an AI-training platform
- an ad network
- an engagement-farming social feed
- a surveillance platform
- a content scraping machine
- a platform designed to harvest user content

## What Federate is for

Federate Network exists to make the web human again. It prioritizes:

- human authorship
- personal websites
- small communities
- creative spaces
- public knowledge
- ownership
- portability
- participant-operated infrastructure
- content preservation
- local-first identity
- a slower, more intentional web

## Defaults that matter

User content is **not used for AI training by default**.

The network is designed around **people creating spaces**, not platforms
extracting attention. Domains belong to people. Content is content-addressed
and portable. Infrastructure is operated by participants, not extracted from
them.

# Federate Network documentation

Documentation lives in per-language folders. No language is the "original":
every page has a twin in every language, cross-linked at the top.

| Language | Docs |
|---|---|
| English | [docs/en-US](en-US/README.md) |
| Português (Brasil) | [docs/pt-BR](pt-BR/README.md) |

Adding a language: copy any existing tree to `docs/<lang>/`, translate, and
add the cross-links. Translations are contributions like any other.

# Documentação da Federate Network

A documentação vive em pastas por idioma. Nenhum idioma é o "original":
toda página tem uma gêmea em cada idioma, com links cruzados no topo.

| Idioma | Docs |
|---|---|
| English | [docs/en-US](en-US/README.md) |
| Português (Brasil) | [docs/pt-BR](pt-BR/README.md) |

Para adicionar um idioma: copie qualquer árvore existente para
`docs/<lang>/`, traduza e adicione os links cruzados. Traduções são
contribuições como qualquer outra.
